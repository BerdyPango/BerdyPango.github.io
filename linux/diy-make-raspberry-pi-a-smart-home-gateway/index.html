背景

通过 HomeAssistant(以下简称 HA) 实现智能家居总控台有多种实现方式，各家也都制作了预定义的系统映像来免去复杂的操作，树莓派非常适合跑过程服务，而 HomeAssistant 也推出了其官方 Docker Image。所以本文主要介绍在 Rasbian 系统上通过 Docker 安装 HomeAssistant。
