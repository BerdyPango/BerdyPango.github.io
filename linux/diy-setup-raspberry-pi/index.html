DIY - 准备树莓派 3b+

Posted on 2018-05-19 | In linux
| Words count in article: 327
记录了搭建新购入的树莓派的全部步骤

背景

新买了个树莓派 3B+，初步设想是准备拿它做 NAS 服务器，HomeAssitant 网关和 KODI 媒体中心。

硬件准备

  • 树莓派 3b+ * 1
  • 网线 * 1
  • Micro SD 卡 * 1，容量大于 8 GB
  • 读卡器 * 1
  • 键盘 * 1
  • HDMI 线 * 1
  • 显示器 * 1
  • USB 电源线 * 1

软件准备

  • RASPBIAN STRETCH LITE 系统映像 -> 下载地址
  • Etcher 映像烧录软件 -> 下载地址

搭建

  1. 使用 Etcher 将 RASPBIAN STRETCH LITE 烧录至 Micro SD 卡中
  2. 将就绪的 Micro SD 卡插入树莓派
  3. 将树莓派接入网线，电源线，HDMI 线和键盘
  4. 树莓派开机，第一次启动会花 1-2 分钟
  5. 使用预设用户 pi/raspberry 登录系统
  6. 键入 sudo raspi-config 进行如下设置:
    1. -> 1 Change User Password，修改 pi 用户登录密码
    2. -> 2 Network Options -> N1 Hostname，更改 Hostname
    3. -> 2 Network Options -> N2 Wi-fi，设置 Wifi
    4. -> 5 Interfacing Options -> P2 SSH，启用 SSH
  7. 重启树莓派
  8. 通过路由器管理界面查询树莓派的 IP 地址，并将该 IP 地址与 MAC 地址绑定为固定 IP 地址。
  9. 撤掉除电源线以外的其他所有线缆，在 PC 上使用 SSH 远程管理树莓派
  10. 执行 apt-get update 以及 apt-get upgrade 命令使预装软件获得最新版本
