Home Assistant 简介

Home Assistant(以下简称 HA) 是一个开源的智能家居网关项目，它可以将市面上所有支持 wifi 接入的智能硬件设备整合到一起进行统一管理，并提供了默认的 Web UI。HA 社区开发了海量组件以支持市面上主流的设备，在 IoT 中扮演了大脑的角色。在个人服务器上架设 HA 有多种实现方式，*UNIX 发行商也都制作了对应的系统映像，HomeAssistant 官方推荐使用「树莓派3B+」作为其宿主机器。

前提条件

为了达到预期效果，最好提前准备好以下设备:

  • 一台安装了 Raspbian 系统的树莓派 3B+，并已接入网络，安装指南可参考「准备树莓派 3b+」
  • 任何支持 wifi 的智能灯具一盏，本文选用了 「Yeelight LED 智能灯」。

安装 Home Assistant

通过 Docker Image 安装及更新一项服务免去了准备其依赖环境的繁琐步骤，官方推出了 HA 的 Docker Image，本文主要介绍通过 Docker 安装 HomeAssistant。如果你不喜欢 Docker，可以参考 Install Home Assistant 其他安装方式。
